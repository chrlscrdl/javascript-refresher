# JavaScript Syntax Refresher

A collection of JavaScript examples and exercises created to review and reinforce fundamental and modern JavaScript concepts.

This project serves as a personal reference and learning resource for practicing JavaScript syntax, understanding core programming concepts, and exploring modern JavaScript features.

## Objectives

- Refresh JavaScript fundamentals
- Practice writing clean and readable code
- Review ES6+ features
- Build a quick reference for common JavaScript syntax and patterns
- Strengthen problem-solving skills through hands-on examples

## Topics Covered

The repository may include examples related to:

### Fundamentals
- Variables (`var`, `let`, `const`)
- Data Types
- Type Conversion
- Operators
- Input and Output
- Comments

### Control Flow
- `if`, `else if`, `else`
- Ternary Operator
- `switch` Statements

### Loops
- `for`
- `while`
- `do...while`
- `for...of`
- `for...in`

### Functions
- Function Declarations
- Function Expressions
- Arrow Functions
- Parameters and Arguments
- Return Values
- Callback Functions

### Arrays
- Creating Arrays
- Array Methods
  - `map()`
  - `filter()`
  - `find()`
  - `reduce()`
  - `forEach()`
- Array Destructuring

### Objects
- Object Creation
- Accessing Properties
- Object Methods
- Object Destructuring
- Spread Operator

### Modern JavaScript (ES6+)
- Template Literals
- Destructuring
- Spread and Rest Operators
- Optional Chaining
- Nullish Coalescing
- Modules
- Classes

### Asynchronous JavaScript
- Promises
- Async/Await
- Error Handling
- Fetch API Examples

## Prerequisites

Before running the examples, make sure you have the following installed:

- Node.js
- npm (comes with Node.js)

Verify your installation:

```bash
node -v
npm -v
```

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd javascript-syntax-refresher
```

## Running the Examples

Run any JavaScript file using Node.js:

```bash
node filename.js
```

Example:

```bash
node variables.js
```

or

```bash
node arrays.js
```
